Drawing With Fourier

Background

For my Math Maker Fair project I decided to explore the math beihind epicycle drawings and learn how to create them. Epicycle drawings are those animations that are sometimes seen online, made from lots of rotating circles that draw an image. While researching for this project, I found a pretty good example of one below that draws Pi, created by 3Blue1Brown.
